## 📌 Sobre o projeto

O **Alquiimia** é uma aplicação web moderna desenvolvida com foco em experiência do usuário, navegação fluida e interface intuitiva.

O sistema segue o conceito **app-like**, oferecendo uma experiência semelhante a aplicativos, com design limpo, responsivo e dinâmico.
